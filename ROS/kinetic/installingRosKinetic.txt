First, to uninstall ros:

$ sudo apt-get purge ros-kinetic

To install (http://wiki.ros.org/kinetic/Installation/Ubuntu):

1. Setup your sources list and keys

$ sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
$ sudo apt-key adv --keyserver hkp://ha.pool.sks-keyservers.net:80 --recv-key 421C365BD9FF1F717815A3895523BAEEB01FA116

2. Ensure debian package index is up-to-date:

$ sudo apt-get update

3. Install

$ sudo apt-get install ros-kinetic-desktop-full

4. Find available packages

$ apt-cache search ros-kinetic

5. Initialize rosdep

$ sudo rosdep init
$ rosdep update

6. Environment setup

$ echo "source /opt/ros/kinetic/setup.bash" >> ~/.bashrc
$ source ~/.bashrc

7. Install rosinstall

$ sudo apt-get install python-rosinstall python-rosinstall-generator python-wstool build-essential

8. Setup default environment sources

$ cd
$ gedit .bashrc

At the end of the file, add:

source /opt/ros/kinetic/setup.bash
source ~/catkin_ws/devel/setup.bash

export ROS_HOSTNAME=10.42.0.1
export ROS_IP=10.42.0.1
export ROS_HOSTNAME=localhost
export ROS_IP=localhost

## Uncomment for local machine
export ROS_MASTER_URI=http://localhost:11311/

## Uncomment for pi
#export ROS_MASTER_URI=http://10.42.0.58:11311/

Save and close

9. Create ROS workspace

$ cd
$ mkdir -p ~/catkin_ws/src
$ cd ~/catkin_ws/
$ catkin_make
